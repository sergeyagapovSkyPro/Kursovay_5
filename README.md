# Курсовая 5. Работа с базами данных

-----
## Описание

В рамках проекта **пользователь** взаимодейсвует с **программой**. Программа получает данные о компаниях и вакансиях с сайта hh.ru, 
проектирует таблицы в БД PostgreSQL и загружает полученные данные в созданные таблицы.

-----
## Основные шаги проекта
**Пользователь**
- Вводит название базы данных
- Вводит название компаний
